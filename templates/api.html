Install the Trident API R package

We are offering an application programming interface for R (http://www.r-project.org) which can be installed using the instruction below.

Use the trident.R script to install the API [TridentAPI version 1.0].

To install a particular package, e.g., TridentAPI, type (or copy and paste) the following in an R command window:
source("http://trident.stjude.org/static/trident.R");
tridentinstall("TridentAPI");
Once installed simply add the following line to the beginning of your scripts.
require("TridentAPI");

Examples of usage:

Get all targets for a microRNA
getTridentdata(microrna="hsa-miR-211-5p", datatype="targets");
Get all microRNAs that target a gene
getTridentdata("TPMT", datatype="gene");
Get a listing of genomes with availble information
getTridentgenomelist();
Get a listing of microRNAs with availble information
getTridentmicroRNAlist();
If you are interested in repackaging TridentAPI on other platforms, please download the source code using this link: Trident API Source Code
