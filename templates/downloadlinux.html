Trident Linux

To simplify setup and use, Trident is available both as a pre-built CentOS 6 system or an ISO to be used to build a CentOS 6 system. The pre-built version is available as an OVA appliance. The virtual machine was built with Virtual Box.

The ISO used to built the system may be downloaded here. This image is the CentOS 6 Install Disk 1 with added packages and kickstart file. The extra packages are located in the extras subdirectory of the image. The image may be burned to a DVD and booted to run the install. In addition to the typical CentOS install, the DVD can, and will by default, run the automatic install used to build the virtual machine listed above. This automatic install will build a CentOS desktop with the extra packages installed. Most notably, this will install the trident executables in the default path as well as Python 2.7. This python may be used by running python2.7 instead of python. Also, the trident web package is located in /var/www/trident_web. However, it is not initially set up to be used, as it requires configuration from the user.
